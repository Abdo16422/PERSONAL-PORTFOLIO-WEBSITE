====================================================
            PERSONAL PORTFOLIO WEBSITE
====================================================

Project Overview
----------------
This project is a fully responsive Personal Portfolio Website developed
using HTML5 and CSS3 only. It showcases personal information, technical skills, 
projects, education, resume, and a contact form.

The website is designed using semantic HTML and modern CSS techniques,
including Flexbox, CSS Grid, Media Queries, and Responsive Design.

----------------------------------------------------

Features
--------
- Responsive Design
- Semantic HTML5 Structure
- Navigation Bar
- Home Section
- About Me Section
- Skills Section
- Projects Section
- Education Table
- Resume Download Button
- Contact Form with HTML Validation
- Footer with Social Links
- Smooth Scrolling
- Button Hover Effects
- Card Hover Effects
- CSS Variables
- CSS Animations
- Google Fonts

----------------------------------------------------

Technologies Used
-----------------
- HTML5
- CSS3
